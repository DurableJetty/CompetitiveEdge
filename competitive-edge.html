Welcome & Instructions

Data Dash Instructions

Objective: Generate as many verified insights as possible within a set time.

  1. Choose a topic area (e.g., Eco-friendly packaging).
  2. Set a timer (20-30 minutes).
  3. Collect as many data-backed insights as you can, verifying each with a second source.
  4. Score = number of confirmed insights.
  5. Try to improve your count in future sessions.

Market Share Domination Instructions

Objective: “Capture” a niche by understanding it deeply and keeping it updated over time.

  1. Select 2-3 market niches.
  2. Establish a baseline understanding with a short research session.
  3. Periodically update your notes with new data to maintain “domination.”
  4. Score over time: more sustained niches = higher score.

Rotating Focus Instructions

Objective: Change research topics daily or periodically, capturing at least one new insight each session.

  1. Create a rotation schedule (e.g., Mon: Competitor Analysis, Tue: Consumer Trends...).
  2. Set a short time block each day for that topic.
  3. Collect one actionable insight per session.
  4. Score daily and see how many consecutive days you maintain quality insights.

Insight Confirmed Instructions

Objective: Each insight must be immediately verified by a second source to count.

  1. Select a broad topic (e.g., DTC food brands).
  2. Set a timeframe (30 mins).
  3. Each insight must be confirmed right away with a reliable source.
  4. Unconfirmed or refuted insights don’t count.
  5. Score at end = total confirmed insights.

Hypothesize & Validate Instructions

Objective: Propose one hypothesis and gather supporting evidence within a limited time, no second chances.

  1. Formulate a single hypothesis (e.g., “A major retailer will introduce a budget-friendly line this quarter”).
  2. Set a timer (30 mins).
  3. Find strong evidence within that time. If successful, you score a point.
  4. If not supported, learn from the attempt and try again another day.
